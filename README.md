## **Installation**

1. Clone the repository:
    ```bash
    git clone https://github.com/KrishBharadwaj394/Chatzyx.git
    ```
2. Navigate to the project directory:
    ```bash
    cd Chatzyx
    ```
3. Install dependencies:
    ```bash
    npm install
    ```
4. Start the development server:
    ```bash
    node app.js
    ```
